Exercice de révision sur le DOM

Révision 2 : Manipulation des classes

Créez le code javascript qui permet de changer l'apparance les éléments du DOM pour configurer un tri. Lorsque l'utilisateur clic sur le nom du tri ou sur la flèche (Par nom, Par date, Par priorité) la flèche haut ou bas s'adapte en fonction de la séquence, ascendant, descendant, ascendant, etc. Les classes CSS fleche-haut et fleche-bas doivent être utilisées. Votre code doit être capable de gérer une liste plus ou moins longue d'items ou bien une liste dynamique.

Méthodes et propriétés pouvant être utiles :
  • addEventListener()
  • événement load, click
  • querySelector(), querySelectorAll()
  • classList (add, remove, contains, etc)
  • parentElement, previousElementSibling, nextElementSibling

Vous ne pouvez pas modifier le code du fichier html. Tout doit se faire par JavaScript, dans le fichier num2.js
