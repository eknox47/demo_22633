Exercice de révision sur le DOM

Révision 1 : Lecture input

Créez le code javascript qui permet à un utilisateur d'entrée un message dans une boite. Pour être valide le message doit comporter entre 20 et 100 caractères. Le bouton Soumettre doit être actif (disabled =false)seulement quand le message est valide. Le nombre de caractères est affiché. Le nombre doit toujours être exact, même au chargement de la page.

Question à se poser : est-ce que les événements change, keyup, keydown ou keypress ont le même comportement?

Méthodes et propriétés pouvant être utiles :
  • addEventListener (événement : load, change, keyup, keydown, keypress)
  • querySelector, querySelectorAll
  • classList (add, remove, contains, etc)
  • Element.disabled

Vous ne pouvez pas modifier le code du fichier html. Tout doit se faire par JavaScript, dans le fichier num1.js

Votre texte contient : caractères
