Exercice sur le DOM

Numéro 2 : Changer l'ordre des items du DOM

Créer le code javascript qui permet de réorganiser l'ordre des éléments du DOM en appuyant sur les flèches (haut et bas). Votre code doit être capable de gérer une liste plus ou moins longue d'items. Et pourquoi pas une liste dynamique d'item?

Méthodes et propriétés utiles :
  • addEventListener()
  • événement load, click
  • querySelector(), querySelectorAll()
  • parentElement, previousElementSibling, nextElementSibling
  • parentElement.insertBefore()

Vous ne pouvez pas modifier le code du fichier html. Tout doit se faire par JavaScript, dans le fichier num2.js
