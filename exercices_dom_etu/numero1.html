Exercice sur le DOM

Numéro 1 : Déclenchements anim css

Créer le code javascript qui permet de démarrer l'animation CSS AnimationDiv lorsqu'on appuie sur le bouton Démarrer animation 1. Pensez à gérer votre interface graphique, une fois que l'animation est lancée que se passe-t-il si j'appuie encore sur le bouton? Est-il nécessaire de pouvoir le faire? Aussi, il n'est pas nécessaire de pouvoir démarrer l'animation plusieurs fois, par contre, si vous voulez le faire, il existe un événement javascript qui est appelé à la fin d'une animation.

Méthodes et propriétés utiles :
  • addEventListener (événement : load, click, animationend)
  • querySelector, querySelectorAll
  • classList (add, remove, contains, etc)

Vous ne pouvez pas modifier le code du fichier html. Tout doit se faire par JavaScript, dans le fichier num1.js
