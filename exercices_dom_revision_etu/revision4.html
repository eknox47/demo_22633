Exercice de révision sur le DOM

Numéro 4 : Numéro 3 + Tri

Créez le code javascript du numéro 3 et y ajouter la possibilité de trier les données selon la valeur de la propriété 'item'. Pour se faire, il faut que l'état des items sont bien conservés. Un item coché, doit resté coché

Méthodes et propriétés pouvant être utiles :
  • DomElement.addEventListener()
  • événement load, click
  • DomElement.querySelector(), DomElement.querySelectorAll()
  • DomElement.innerHTML
  • DomElement.insertAdjacentHTML()
  • boucle Array.forEach ou for...in ou for...of ou simplement for
  • Array.findIndex
  • Array.sort

Vous ne pouvez pas modifier le code du fichier html. Tout doit se faire par JavaScript, dans le fichier num4.js
