Exercice de révision sur le DOM

Numéro 3 : Création d'une liste à partir de données dynamique

Créez le code javascript qui permet de générer la liste d'items à partir des données fournies dans le tableau présent dans le fichier num3.js. La liste d'items doit être basée sur l'exemple dans la section liste-item. Votre code doit traiter les données et s'adapter si le nombre d'items change. Si un item à une propriété barre == true, vous devez cocher le checkbox et ajouter la classe barre sur div.item

Si le coeur vous en dit, programmez le fonctionnement des flèche (haut et bas) afin que les items se déplace dans le DOM.

Et pourquoi ne pas créer la fonction qui permet de lier l'état de la case à cocher et l'apparance de l'item. Quand l'utilisateur coche le "checkbox", vous devez ajouter la classe "barre" sur div.item, quand il décoche la case, la classe barre est retirée.

Bonus (ou pas) : Vous devez adapter l'état du tableau de données pour correspondre à l'état de l'affichage.

Méthodes et propriétés pouvant être utiles :
  • DomElement.addEventListener()
  • événement load, click
  • DomElement.querySelector(), DomElement.querySelectorAll()
  • DomElement.innerHTML
  • DomElement.insertAdjacentHTML()
  • boucle Array.forEach ou for...in ou for...of ou simplement for
  • Array.findIndex

Vous ne pouvez pas modifier le code du fichier html. Tout doit se faire par JavaScript, dans le fichier num3.js
